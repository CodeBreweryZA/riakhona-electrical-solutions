# Riakhona Electrical Solutions

A professional business website for an electrical services company, built with HTML and CSS.

## Sections

- **About** – Company background and mission
- **Services** – Electrical installation, maintenance, and repair offerings
- **Gallery** – Images of completed projects (pic1.jpg–pic4.jpg)
- **Contact Form** – Enquiry form for prospective clients
- **Navigation** – Sticky header with smooth anchor links
- **Footer** – Contact details and social links

## Features

- Fully responsive design (mobile-friendly)
- Modern colour scheme with CSS custom properties
- Font Awesome icon integration

## Getting Started

Open `index.html` in any modern web browser. No server or build tools required.

## Technologies Used

- HTML5
- CSS3
